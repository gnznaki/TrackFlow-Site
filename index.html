Features
Everything you need to stay on top of your work
Built for the way producers actually work — not how project managers think they do.
Auto-scan DAW files
Point TrackFlow at your drives and it finds every .flp, .als, .ptx, .rpp file automatically. No manual importing.
Visual kanban boards
Drag projects between columns — Idea, In Progress, Mixing, Done — or define your own stages.
Notes & custom tags
Add session notes, BPM, key, client info — whatever you need. Tag and filter across all your projects.
One-click open in DAW
Click any card to open the project file directly in FL Studio, Ableton, Pro Tools, or Reaper. No finder, no file browser.
Cloud sync
Your board stays in sync across every machine you work on. Switch computers without losing your place. Premium
Real-time collaboration
Share boards with producers, engineers, or A&R. Everyone sees changes in real-time with role-based access. Premium

Installation guide

Security & trust
⚠ You will always see a Windows security warning

This is completely normal and expected. TrackFlow is not yet signed with a paid Windows code-signing certificate. Microsoft SmartScreen warns about any app that hasn't built up a reputation through a certificate authority — including legitimate, safe software.

This does not mean the app is dangerous. It means it hasn't paid Microsoft for a certificate yet. Clicking "More info" → "Run anyway" is the correct action.

Every release is scanned before publishing The v1.2.1 VirusTotal scan shows 68 of 71 engines clean. View the full results →

The 3 flagging engines (Sophos, SecureAge, Arctic Wolf) use heuristic ML detection only — none match a known malware signature. This is a documented false positive pattern for newly compiled, unsigned apps. Every major engine — Windows Defender, Kaspersky, ESET, Bitdefender, Avast, Norton, Malwarebytes — reports clean.
