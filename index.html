Security & trust
⚠ You will always see a Windows security warning

This is completely normal. TrackFlow is not yet signed with a paid Windows code-signing certificate. Microsoft SmartScreen warns about any app that hasn't built reputation through a certificate authority.

This does not mean the app is dangerous. Clicking "More info" → "Run anyway" is the correct action.

Every release is scanned before publishing The v1.2.1 VirusTotal scan shows 68 of 71 engines clean. View the full results →

The 3 flagging engines (Sophos, SecureAge, Arctic Wolf) use heuristic ML detection only — none match a known malware signature. Every major engine — Windows Defender, Kaspersky, ESET, Bitdefender, Avast, Norton, Malwarebytes — reports clean.
