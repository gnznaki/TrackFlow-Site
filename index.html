Installation guide
1
Download the installer Click the button above. GitHub will download a .exe setup file directly to your computer.
2
Run the installer Double-click the downloaded file. Windows may show a SmartScreen security prompt — this is expected (see the note on the right).
3
Click "More info" → "Run anyway" On the SmartScreen popup, click "More info" at the bottom left, then click "Run anyway". TrackFlow will install normally.
4
You're in TrackFlow launches automatically after install. No login required — just point it at your project folders and start organizing.
Security & trust
⚠ You will always see a Windows security warning

This is completely normal and expected. TrackFlow is not yet signed with a paid Windows code-signing certificate. Microsoft SmartScreen warns about any app that hasn't built up a reputation through a certificate authority — including legitimate, safe software.

This does not mean the app is dangerous. It means it hasn't paid Microsoft for a certificate yet. Clicking "More info" → "Run anyway" is the correct action.

Every release is scanned before publishing We run each build through VirusTotal before it's released. You can independently verify any release by uploading the installer to VirusTotal yourself — it will confirm the file is clean.
